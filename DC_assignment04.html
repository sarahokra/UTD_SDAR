a. Modify the program to scrape other tables in Wikipedia

In this assignment, I scraped the Pretty Cure (光之美少女) information from Wikipedia. I extracted the series list tables from both the Traditional Chinese and English Wikipedia pages, then integrated the information from both sources into a single table/dataframe.

  • Chinese Wikipedia: https://zh.wikipedia.org/wiki/光之美少女系列

  • English Wikipedia: https://en.wikipedia.org/wiki/Pretty_Cure

The Chinese page contains detailed broadcast period information in the format “YYYY年MM月DD日”, while the English page provides series titles, generation, episodes, and director information. By merging both sources, I created a comprehensive dataset that includes English titles with properly formatted start and end dates.

c. Suggest a data plan to acquire web data for research.

To systematically collect and update data for research, I would follow these steps:

1. Identify reliable data sources such as Wikipedia, government open data portals, or official databases related to the research topic.

2. Use R web-scraping packages like rvest or APIs through httr to extract tables or JSON data automatically.

3. Clean and structure the dataset by removing redundant rows/columns, converting date variables, and standardizing variable names.

4. Store the cleaned data in a local CSV file or database (e.g., SQLite) for analysis.

5. Automate the process using scheduled R scripts to regularly update the dataset.

6. Apply the dataset in research — for example, analyzing long-term trends in anime production or cultural media evolution.
